BootyPrint CSS Variables Guide

This guide explains how to customize BootyPrint using CSS custom properties (CSS variables). BootyPrint allows you to override its default styling without modifying the SCSS source files.

How to Use CSS Variables with BootyPrint

Simply define your custom CSS variables in the :root selector in your HTML file or in a separate CSS file. These will override the default SCSS variables used by BootyPrint. 

:root {
    --primary: #3f51b5;
    --secondary: #2196f3;
    --font-size-base: 14px;
    --page-size: A5 landscape;
}

Example

Here's an example of customizing BootyPrint using CSS variables:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Customized BootyPrint Example</title>
    <link rel="stylesheet" href="path/to/bootyprint.css">
    <style>
        :root {
            --primary: #3f51b5;
            --secondary: #2196f3;
            --font-size-base: 14px;
            --page-size: A5 landscape;
            --font-family-base: 'Georgia', serif;
            --border-radius: 0.5rem;
        }
    </style>
</head>
<body>
    <!-- Your content here -->
</body>
</html>

This setup will override the default primary and secondary colors, change the base font size to 14px, set the page size to A5 landscape, use Georgia as the base font, and make border radiuses larger.
